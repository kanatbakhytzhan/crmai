1️⃣  Откройте PowerShell в этой папке и выполните:

    python -m venv venv
    .\venv\Scripts\Activate.ps1

    (Если ошибка прав: Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser)


2️⃣  Установите зависимости:

    pip install -r requirements.txt


3️⃣  Запустите сервер:

    python main.py

    (Или двойной клик на run.bat)

После запуска откройте в браузере:

📍 Swagger UI (API документация):
   http://localhost:8000/docs

📍 Главная страница:
   http://localhost:8000

📍 Health Check:
   http://localhost:8000/api/health


Протестируйте через Swagger:
1. Откройте POST /api/chat
2. Нажмите "Try it out"
3. Введите:
   - user_id: test_user_1
   - message_text: Здравствуйте, хочу построить дом
4. Нажмите "Execute"

Бот должен ответить на русском и начать квалификацию! 🎉

1. Найдите вашего бота в Telegram (токен из .env)

4. Лид придет в Telegram с кнопками управления!

Q: Как изменить имя бота?
A: app/services/openai_service.py → SYSTEM_PROMPT

Q: Как добавить новые вопросы?
A: app/services/openai_service.py → SYSTEM_PROMPT

Q: Как протестировать аудио?
A: Swagger UI → POST /api/chat → загрузите .ogg/.mp3/.wav

Q: Где хранятся данные?
A: SQLite файл sales_bot.db (создается автоматически)

Q: Как просмотреть историю диалогов?
A: GET /api/user/{user_id}/history
